Welcome to the East River Resiliency project, presented by the East River Collaborative. This tool allows you to view and compare different East River flooding scenarios in and around Green Bay, Wisconsin.

How to Use

Select a Display:

Select how you want the flood to be displayed. Extent displays the footprint of the flood, while Depth displays the flood depth.



Select a Lake Level:

The extent and depth of flooding depends on the level of Lake Michigan. High will show flood levels when the lake is high, Low will show flood levels when the lake is low.



Select a Flood Level

Select one or more flooding scenarios. These scenarios are ranked in order of severity, with a 1-year flood being the least severe, and a 500 year flood being the most severe.



Add Overlays

You are also able to add some supplementary data to the map, including the HAZUS dataset, which shows which buildings will be affected by various flooding scenarios, and the Social Vulnerability Index (SVI) for census block groups along the river.
